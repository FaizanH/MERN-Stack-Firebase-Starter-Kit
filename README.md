# MERN-Stack-Firebase-Starter-Kit

A template I've put together to help get up and running with MERN stack on Google Firebase.
[^1]: React.JS runs on Firebase Hosting while the server-side Node.JS, Express.JS, MongoDB run in Firebase Functions.

### Access Points

    (Local)
    http://localhost:5001/<project-name>/<region>/<function-name>
    
    (Staging)
    https://<project-name>.firebaseapp.com

### End Points
    /
    /admin
    /other
